# Battleship

Based on [Battleship (game)](https://en.wikipedia.org/wiki/Battleship_%28game%29)

## Run

```bash
git clone https://github.com/Tucker-Eric/battleship.git
cd battleship
#  Copy the .env file
cp .env.dist .env
# To run in production mode
make
# To run in development mode
make dev
```

## Stop Service

```bash
make down
```
